# ArchDistribution

보고서용 문화유적분포지도 제작을 단순화하는 QGIS 플러그인입니다.
지형도 병합부터 유적 목록 정리까지 복잡한 과정을 자동화합니다.

## ✨ 주요 기능
## ✨ 주요 기능
- **지형도 관리**: 지형도 병합 및 스타일(0.05mm) 자동 적용
- **분석 자동화**: 버퍼(300m/500m) 생성, 축척별 도곽 계산
- **유적 정리**: 주변 유적 통합, 거리순 번호, 스마트 네이밍
- **데이터 반출**: 일러스트레이터용 레이어 및 엑셀 목록 생성
- **실시간 로그**: 작업 진행 상황 실시간 확인
