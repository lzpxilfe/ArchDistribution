# ArchDistribution (Pro v1.3.3)

**문화유적분포지도 자동 생성 & 스마트 분류 엔진**

이 플러그인은 고고학 연구자와 조사기관을 위해 개발된 QGIS 전용 자동화 도구입니다. 복잡한 수치지형도 병합, 유적 분류, 도곽 설정, 스타일링 작업을 단 한 번의 클릭으로 해결합니다.

---

## ✨ 1.3.3 버전의 강력한 새로운 기능

### 🧠 1. 스마트 필터링 & 정밀 분류 (Smart Classification)
단순한 DB 매칭을 넘어, 유적의 **명칭(Keyword)**을 분석하여 숨겨진 성격을 찾아냅니다.
*   **유물 정밀 포착**: 'OO사지'로 등록된 유적이라도, 이름에 '석불', '광배'가 포함되면 **'불교석조물'**로 자동 재분류합니다.
*   **분묘 세분화**: 모든 무덤을 똑같이 취급하지 않습니다.
    *   **지석묘(고인돌)**: 청동기시대 지석묘
    *   **고분(옛무덤)**: 역사적 가치가 있는 고대 고분
    *   **분묘(일반)**: 민묘 등 일반 무덤
*   **제외 제안 목록 (Exclusion Candidates)**: '입회조사', '참관', '보호수' 등 유적이 아닌 항목을 자동으로 감지하여 **제외 목록**에 띄워줍니다. (체크 해제 시 복구 가능)

### 🎨 2. 최적화된 사용자 인터페이스 (UI)
*   **전체 스크롤 지원**: 노트북이나 작은 화면에서도 잘리지 않도록 전체 화면 스크롤을 지원합니다.
*   **로그 공간 최적화**: 작업 로그창이 화면을 과도하게 차지하지 않도록 개선했습니다.

## 🚀 주요 기능 (Core Features)

### 1. 지도 제작 자동화
*   **수치지형도 자동 병합**: 수십 장의 DXF/NGI 지형도를 분석 범위에 맞춰 자동으로 자르고 병합합니다.
*   **버퍼 분석 (Buffer)**: 조사구역 기준 50m, 100m 등 원하는 반경을 즉시 그려줍니다.
*   **도곽 자동 설정**: 축척(1:5000 등)과 용지 크기(A4, 보고서용)에 맞춰 최적의 지도 범위를 계산합니다.

### 2. 데이터 정리 및 반출
*   **유적 번호 부여**: '북→남', '거리순', '가나다순' 등 원하는 규칙으로 유적 번호를 자동 부여합니다.
*   **지능형 네이밍**: '유적명 없음' 데이터도 지번이나 위치를 추적하여 임시 명칭을 생성합니다.
*   **PDF/Excel 반출**: 보고서 삽입용 고화질 PDF와 유적 목록 엑셀 파일을 생성합니다.

## 🛠 설치 및 사용 방법

### 설치 (Installation)
1. QGIS를 실행하고 [설정] -> [사용자 프로필] -> [프로필 폴더 열기]를 클릭합니다.
2. `python/plugins` 폴더로 이동합니다.
3. 다운로드 받은 `ArchDistribution` 폴더를 통째로 복사해 넣습니다.
4. QGIS를 재시작하거나 [플러그인 관리]에서 `ArchDistribution`을 체크합니다.

### 사용 순서 (Workflow)
1. **레이어 준비**: QGIS에 조사구역(Polygon), 수치지형도(Line), 주변 유적(Point/Polygon) 레이어를 띄웁니다.
2. **플러그인 실행**: 툴바의 🏛 아이콘을 클릭합니다.
3. **입력 선택**: 조사구역과 지형도, 유적 레이어를 선택합니다.
4. **[속성 분류 실행] 클릭**: 스마트 엔진이 유적의 시대와 성격을 분석합니다.
    *   *Tip: '제외 제안 목록'에 뜬 항목 중 살려야 할 유적이 있는지 확인하세요.*
5. **[▶ 분석 및 지도 생성 실행] 클릭**: 모든 작업이 자동으로 수행됩니다.
