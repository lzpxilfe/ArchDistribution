# ArchDistribution

보고서용 문화유적분포지도를 쉽게 만들 수 있게 해주는 QGIS 플러그인입니다.

## 🌟 주요 기능
- **수치지형도 병합 및 스타일 자동 적용**: 여러 매의 수치지형도를 하나로 합치고, 보고서용 스타일(0.05mm 검정색 선)로 자동 변경합니다.
- **조사구역 버퍼 자동 생성**: 300m, 500m 등 원하는 거리의 버퍼를 한 번에 생성합니다.
- **도곽 및 영역 자동 계산**: 출력 판형(A4, A3 등)과 축척에 맞춰 실제 지도상의 도곽 영역을 계산하고 생성합니다.
- **주변 유적 통합 및 번호 부여**: 영역 내의 유적 데이터를 수집/병합하고, 조사구역 중심에서의 거리에 따라 자동으로 번호를 매깁니다.
- **엑셀 내보내기 최적화**: 번호, 유적명, 주소, 면적 정보가 포함된 표준화된 속성 테이블을 생성하여 보고서 작성을 돕습니다.
- **실시간 작업 로그**: 플러그인 내 로그 창을 통해 작업 진행 상황을 실시간으로 확인할 수 있습니다.
